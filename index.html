Política de Privacidade

Invisi Mobile - Aplicativo de Monitoramento de Transportes

Última atualização: 04 de dezembro de 2025

1. Introdução

Bem-vindo ao Invisi Mobile. Esta Política de Privacidade descreve como coletamos, usamos, armazenamos e protegemos suas informações pessoais quando você utiliza nosso aplicativo móvel disponível para Android e iOS.

Ao utilizar o Invisi Mobile, você concorda com os termos descritos nesta política. Caso não concorde com algum ponto, recomendamos que não utilize o aplicativo.

Informações do Aplicativo:
  • Nome: Invisi-app
  • Versão: 1.0.4
  • Identificador: com.invisi.app
  • Plataformas: Android e iOS

2. Dados Coletados

O Invisi Mobile coleta e processa as seguintes categorias de dados para fornecer os serviços de monitoramento e gestão de transportes:

2.1. Dados de Autenticação

Para permitir seu acesso seguro ao sistema, coletamos:

  • Nome de usuário: Identificador único fornecido pela sua organização
  • Senha: Credencial de acesso (armazenada de forma segura e criptografada)
  • Token de autenticação: Gerado automaticamente após login bem-sucedido
  • Perfil/Função do usuário: Grupo de permissões associado à sua conta
  • Filial/Lotação: Unidade organizacional à qual você está vinculado

2.2. Dados Operacionais

Durante o uso do aplicativo, processamos informações relacionadas às suas atividades:

  • Filial selecionada: Unidade que você está monitorando atualmente
  • Preferências de filtros: Configurações personalizadas de visualização (serviços, operações, grupos, produtos)
  • Dados de monitoramento: Informações sobre veículos, cargas, descargas e operações logísticas

2.3. Dados de Sessão

  • Última filial acessada: Para facilitar o acesso em sessões futuras
  • Metadata de autenticação: Data de criação do token, tipo e última validação
  • Cache temporário: Dados armazenados para melhorar a performance do aplicativo
Tipo de Dado Finalidade Tempo de Retenção
Credenciais de login Autenticação e controle de acesso Durante a sessão ativa
Token de autenticação Manter sessão segura Até 24 horas ou logout
Dados operacionais Monitoramento e gestão Cache de 3-10 minutos
Preferências do usuário Personalização da experiência Até o logout

3. Finalidade do Uso dos Dados

Os dados coletados são utilizados exclusivamente para:

  1. Autenticação e Segurança: Verificar sua identidade e garantir acesso seguro ao sistema
  2. Funcionalidades do Aplicativo: Fornecer serviços de monitoramento de transportes, veículos, cargas e operações logísticas
  3. Controle de Acesso: Gerenciar permissões baseadas em seu perfil e filial
  4. Melhoria de Experiência: Otimizar o desempenho através de cache e personalização de preferências
  5. Segurança Operacional: Prevenir acessos não autorizados e ataques ao sistema
Importante: Não utilizamos seus dados para fins publicitários, marketing de terceiros ou qualquer propósito além dos mencionados acima.

4. Armazenamento e Proteção de Dados

4.1. Local de Armazenamento

Todos os dados são armazenados localmente no seu dispositivo móvel. Não realizamos backup automático em nuvem ou servidores externos de terceiros.

4.2. Métodos de Proteção

Dados Sensíveis - Armazenamento Criptografado:

As seguintes informações são protegidas com criptografia via Expo SecureStore:

  • Token de autenticação
  • Metadata do token
  • Nome de usuário
  • Perfil do usuário
  • Filial do usuário

Dados Não-Sensíveis - AsyncStorage:

  • Cache de dados operacionais
  • Preferências de filtros
  • Última filial selecionada

4.3. Medidas de Segurança Implementadas

  • Validação e Sanitização: Todas as entradas de usuário são validadas e sanitizadas para prevenir ataques de injeção
  • Proteção contra SQL Injection: Filtros automáticos detectam e bloqueiam padrões maliciosos
  • Rate Limiting: Máximo de 5 tentativas de login em 15 minutos, com bloqueio temporário de 30 minutos após exceder o limite
  • Timeout de Sessão: Sessões inativas são encerradas automaticamente por segurança
  • Validação de Tokens: Tokens JWT são verificados quanto à expiração e integridade
  • Limpeza Segura: Todos os dados são completamente removidos do dispositivo ao fazer logout
Aviso de Segurança: A comunicação entre o aplicativo e os servidores da empresa utiliza protocolo HTTP. Embora implementemos diversas camadas de segurança no aplicativo, recomendamos cuidado ao usar redes Wi-Fi públicas.

5. Compartilhamento de Dados

5.1. Servidores Próprios da Empresa

O aplicativo se comunica exclusivamente com os servidores internos da empresa, localizados nas seguintes filiais:

  • Filial LDA
  • Filial CHP
  • Filial FND
  • Filial NMD
  • Filial NMG

5.2. Ausência de Serviços de Terceiros

O Invisi Mobile NÃO compartilha dados com:

  • Serviços de análise de terceiros (Google Analytics, Firebase Analytics, etc.)
  • Redes sociais (Facebook, Twitter, LinkedIn, etc.)
  • Plataformas de publicidade
  • Serviços de crash reporting externos
  • Serviços de backup em nuvem de terceiros
  • Qualquer outra entidade externa à organização
Garantia de Privacidade: Seus dados permanecem exclusivamente dentro da infraestrutura da empresa e são usados apenas para as finalidades operacionais descritas nesta política.

6. Permissões do Aplicativo

O Invisi Mobile solicita apenas as permissões essenciais para seu funcionamento:

6.1. Android

  • INTERNET: Necessária para comunicação com os servidores da empresa
  • ACCESS_NETWORK_STATE: Permite verificar a disponibilidade de conexão à internet

6.2. iOS

  • Acesso à Internet: Permissão padrão para comunicação de rede

Não solicitamos acesso a: Câmera, microfone, localização GPS, contatos, galeria de fotos, calendário ou qualquer outro dado pessoal do dispositivo.

7. Retenção de Dados

Os prazos de retenção variam conforme o tipo de dado:

Tipo de Dado Período de Retenção
Credenciais de autenticação Durante a sessão ativa ou até logout manual
Token de autenticação (JWT) Até expiração definida no token
Token simples Máximo de 24 horas
Cache de dados operacionais 3 a 10 minutos (dependendo do tipo)
Preferências de filtros Até logout ou desinstalação do app

Ao fazer logout, todos os dados são imediatamente removidos do dispositivo, incluindo credenciais, tokens, cache e preferências.

8. Direitos do Usuário

Como usuário do Invisi Mobile, você possui os seguintes direitos:

8.1. Direito de Acesso

Você tem o direito de acessar os dados pessoais armazenados no aplicativo durante sua sessão ativa.

8.2. Direito de Exclusão

Você pode excluir todos os seus dados do dispositivo a qualquer momento através da função de logout. Alternativamente, a desinstalação do aplicativo também remove todos os dados locais.

8.3. Direito de Retificação

Para alterar informações como nome de usuário, senha ou perfil, entre em contato com o administrador do sistema em sua organização.

8.4. Direito de Portabilidade

Como os dados são processados localmente e vinculados ao sistema da empresa, a portabilidade deve ser solicitada diretamente ao departamento responsável.

8.5. Direito de Reclamação

Caso tenha preocupações sobre como seus dados são tratados, você pode entrar em contato através dos canais indicados na seção de contato.

9. Uso por Menores de Idade

O Invisi Mobile é destinado exclusivamente para uso profissional e corporativo. Não coletamos intencionalmente dados de menores de 18 anos. Se você é menor de idade, não deve utilizar este aplicativo sem autorização e supervisão de um responsável legal.

10. Alterações nesta Política

Esta Política de Privacidade pode ser atualizada periodicamente para refletir mudanças em nossas práticas ou requisitos legais. Recomendamos que você revise esta página regularmente.

Alterações significativas serão comunicadas através de notificação no aplicativo ou por outros meios apropriados.
